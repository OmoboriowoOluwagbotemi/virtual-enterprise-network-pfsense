# Virtual Enterprise Network Design & Security Lab

## Executive Summary
This project demonstrates the design and implementation of a secure,
segmented enterprise network using pfSense, VLANs, DHCP, DNS, and
Wireshark for traffic validation. The environment is deployed in
VirtualBox and follows Zero Trust networking principles.

## Technologies Used
- pfSense
- VirtualBox
- Ubuntu Server 22.04
- Kali Linux
- Wireshark
- VLANs (802.1Q)
- DHCP, DNS, ICMP

## Project Scope
- VLAN segmentation (HR, IT, Guest)
- Firewall policy enforcement
- DHCP and DNS services
- Packet capture and analysis
